# Weather Bot PWA

Files included:
- index.html
- manifest.webmanifest
- sw.js
- icon-192.png
- icon-512.png

How to use:
1. Upload all files to the same folder on any static host.
2. Open over HTTPS in Chrome mobile.
3. Use the "Install app" button or Chrome's Add to Home Screen flow.

Good hosting options:
- GitHub Pages
- Netlify
- Vercel

Notes:
- PWA installability requires HTTPS or localhost.
- Weather data: Open-Meteo
- Map: OpenStreetMap / Leaflet
- Reverse geocoding: Nominatim
